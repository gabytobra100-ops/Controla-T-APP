# 📱 ControlaT

ControlaT es una aplicación móvil desarrollada en Android que permite gestionar de manera eficiente un negocio pequeño, incluyendo el control de productos, clientes, ventas y reportes.

---

## 🚀 Funcionalidades principales

- 📦 Gestión de productos (agregar, editar, eliminar)
- 👥 Gestión de clientes
- 💰 Registro de ventas
- 📊 Reportes automáticos:
  - Total de ventas
  - Ingresos generados
  - Cliente más frecuente
  - Producto más vendido
  - Alertas de stock

---

## 🧠 Tecnologías utilizadas

- Java
- Android Studio
- Room Database
- RecyclerView
- Material Design

---

## 📱 Características

- Interfaz moderna y amigable
- Uso de base de datos local (Room)
- Sistema adaptable a distintos tipos de negocio
- Flujo completo de gestión (de producto a venta)

---

## 💡 Objetivo

Esta aplicación fue desarrollada como un sistema de apoyo para pequeños emprendedores, permitiéndoles llevar un control básico pero funcional de su negocio.
